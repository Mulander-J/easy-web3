# WEB3 Starter

MonoRepo for bunch of Web3 DApp bolierplates.

## Tech Stack

**Language:** `JS + TS`, `Solidity`, `Rust`, `Motoko`, `Cadence`, `Cairo`

**Framework:** `React`, `Vue`, `Hardhat`, `Anchor`, `DFX`, `Scarb`
